# Anime Next - 安裝指南

> Chrome 擴充功能安裝說明

**版本**: v1.0.0
**更新日期**: 2025-11-06

---

## 📦 壓縮檔安裝方式

### 方法 1: 使用壓縮檔安裝（推薦）

#### 步驟 1: 解壓縮檔案

1. 找到 `anime-next-v1.0.0.zip` 檔案
2. 右鍵點擊 → 選擇「解壓縮全部」
3. 選擇一個**永久存放**的位置（不要放在桌面或下載資料夾）
   - 建議位置：`C:\Program Files\ChromeExtensions\anime-next\`
   - 或者：`C:\Users\您的使用者名稱\AppData\Local\ChromeExtensions\anime-next\`

⚠️ **重要**：解壓縮後的資料夾**不能刪除或移動**，否則擴充功能會失效！

#### 步驟 2: 在 Chrome 中載入擴充功能

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入：
   ```
   chrome://extensions/
   ```
3. 在右上角**開啟**「開發人員模式」開關
4. 點擊左上角的「**載入未封裝項目**」按鈕
5. 選擇您剛才解壓縮的資料夾
6. 點擊「選擇資料夾」

#### 步驟 3: 驗證安裝

安裝成功後，您應該會看到：

```
✅ Anime Next - 鍵盤導航
   版本 1.0.0
   狀態：已啟用
   ID: [一串隨機字符]
```

## 🎮 開始使用

### 1. 開啟測試頁面

在 Chrome 中開啟任何有分頁的網頁，例如：
- [DataTables 範例](https://datatables.net/examples/basic_init/zero_configuration.html)
- 或任何有分頁按鈕的網站

### 2. 測試功能

1. **確保焦點不在輸入框中**（點擊頁面空白處）
2. 按下鍵盤 **右方向鍵 (→)** → 切換到下一頁
3. 按下鍵盤 **左方向鍵 (←)** → 切換到上一頁

### 3. 開啟設定

1. 點擊工具列的擴充功能圖示（拼圖圖示）
2. 找到並點擊 "Anime Next - 鍵盤導航"
3. 設定彈出視窗會開啟

**可用設定**：
- ✅/❌ 啟用鍵盤導航
- ✅/❌ 視覺回饋效果
- 🔄 重置設定
- 🧪 開啟測試頁面

## 🔧 除錯與故障排除

### 問題 1: 載入時出現錯誤

**可能原因**：檔案損壞或缺失

**解決方法**：
1. 重新下載 `anime-next-v1.0.0.zip`
2. 重新解壓縮
3. 在 `chrome://extensions/` 點擊「重新載入」

---

### 問題 2: 功能沒有反應

**檢查步驟**：
1. 確認擴充功能狀態為「已啟用」
2. 重新整理網頁
3. 按 F12 開啟開發者工具
4. 切換到 Console 分頁
5. 檢查是否有以下訊息：
   ```
   [Anime Next] 開始初始化...
   [Anime Next] 鍵盤導航已啟用
   ```

如果沒有看到這些訊息：
- 重新整理頁面
- 或在 `chrome://extensions/` 點擊「重新載入」擴充功能

---

### 問題 3: 在某些網站上無法使用

**可能原因**：該網站使用不同的分頁結構

**目前支援的網站類型**：
- DataTables 分頁
- 使用標準 HTML 分頁按鈕的網站

**未來更新將支援**：
- Bootstrap Pagination
- Material UI Pagination
- 更多自訂分頁組件

## 📋 功能說明

### 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| ← (左方向鍵) | 上一頁 |
| → (右方向鍵) | 下一頁 |

### 智能排除

擴充功能會自動排除以下情況，不觸發分頁切換：
- ✅ 在 `<input>` 輸入框中
- ✅ 在 `<textarea>` 文字區域中
- ✅ 在 `<select>` 下拉選單中
- ✅ 在可編輯元素 (contenteditable) 中

### 視覺回饋

啟用視覺回饋後，按鈕被點擊時會顯示：
- 💫 光暈擴散動畫
- 🔵 脈衝效果

---

## 🔄 更新擴充功能

當有新版本時：

1. 下載新的 `anime-next-vX.X.X.zip`
2. 解壓縮到**同一個位置**（覆蓋舊檔案）
3. 前往 `chrome://extensions/`
4. 找到 Anime Next 擴充功能
5. 點擊「**重新載入**」按鈕

您的設定會自動保留（儲存在 Chrome Sync）。

---

## ❌ 解除安裝

如果您想移除擴充功能：

1. 前往 `chrome://extensions/`
2. 找到 Anime Next 擴充功能
3. 點擊「**移除**」按鈕
4. 確認移除
5. （可選）刪除解壓縮的資料夾

## 📦 檔案結構

壓縮檔包含以下檔案：

```
anime-next/
├── manifest.json         # 擴充功能配置檔
├── content.js           # 核心功能腳本
├── background.js        # 背景服務
├── popup.html           # 設定介面 UI
├── popup.js             # 設定介面邏輯
├── styles/
│   └── content.css      # 視覺效果樣式
├── icons/
│   └── next-prev.ico    # 擴充功能圖示
└── README.md            # 專案說明
```

**總大小**: 約 12 KB
