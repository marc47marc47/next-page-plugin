⚡ 5 步驟快速安裝

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 1️⃣  開啟 Edge 擴充功能頁面

   在 Edge 網址列輸入：edge://extensions/

   或執行：開啟Edge擴充功能頁面.bat

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 2️⃣  開啟開發人員模式

   在左下角找到「開發人員模式」開關
   將它打開（變成藍色）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 3️⃣  點擊「載入解壓縮」按鈕

   ⭐ 重要！您要找的按鈕是「載入解壓縮」

   開啟開發人員模式後，應該會看到三個按鈕：

   ┌─────────────────┐
   │  載入解壓縮     │ ← ✅ 點這個！
   └─────────────────┘

   ┌─────────────────┐
   │  套件擴充功能   │ ← ❌ 不要點這個！
   └─────────────────┘

   ┌─────────────────┐
   │  更新           │
   └─────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 4️⃣  選擇擴充功能資料夾

   點擊「載入解壓縮」後，會彈出資料夾選擇視窗

   📁 請選擇：C:\Users\marc4\js\anime-next

   ⚠️  重點：
   - 直接選擇這個資料夾
   - 不要進入資料夾裡面
   - 不要找 .zip 檔案
   - 選擇資料夾本身就對了

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

步驟 5️⃣  確認安裝成功

   安裝成功後，您應該會看到：

   ┌─────────────────────────────────────────┐
   │ ✅ Anime Next - 鍵盤導航                │
   │                                         │
   │ 版本 1.0.0                              │
   │ ID: [一串隨機字符]                      │
   │                                         │
   │ 使用鍵盤左右鍵控制網頁上一頁/下一頁    │
   │                                         │
   │ [詳細資料] [移除] [錯誤]               │
   └─────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎉 安裝完成！開始測試

現在您可以：

1. 開啟任何有分頁的網頁
2. 按下鍵盤的 ← 或 → 方向鍵
3. 測試分頁切換功能

建議測試網站：
https://datatables.net/examples/basic_init/zero_configuration.html

❓ 常見問題

Q: 為什麼按鈕叫「載入解壓縮」？
A: 這是 Edge 中文版的翻譯，英文版叫 "Load unpacked"
   就是「載入未封裝的擴充功能」的意思

Q: 我需要先解壓縮 zip 檔案嗎？
A: 不需要！直接選擇 anime-next 資料夾就可以了

Q: 「套件擴充功能」和「載入解壓縮」有什麼差別？
A: 「載入解壓縮」是給開發者用的，適合個人使用
   「套件擴充功能」是打包給別人安裝用的，會產生 .crx 檔案

Q: 如果看到「私密金鑰已存在」的錯誤怎麼辦？
A: 那是因為您點了「套件擴充功能」
   請改用「載入解壓縮」就不會有這個問題了
